# WasteZero - Smart Waste Management System

## 🚀 Project Overview
WasteZero is a full-stack web application that helps manage waste collection, volunteer opportunities, and NGO coordination.

## 🛠 Tech Stack
- Frontend: Angular
- Backend: Node.js, Express
- Database: MongoDB

## ✨ Features
- User Authentication (JWT)
- Waste Request Management
- Opportunity Matching
- Admin Dashboard

## ▶️ Run Locally

### Frontend
npm install
ng serve

### Backend
cd wastezero-backend
npm install
npm start
